# YouTube Video Downloader (YVD)

The YouTube Video Downloader (YVD) is a simple Python application that allows users to download videos from YouTube. It utilizes the `pytube` library to fetch and download videos in the highest available resolution.

## Features

- **Simple Interface**: Uses a command-line interface for inputting the YouTube video URL and selecting the save location.
- **High-Resolution Downloads**: Downloads videos in the highest resolution available.
- **Easy to Use**: Minimal setup required to start downloading videos.

## Getting Started

### Prerequisites

- Python 3.x
- `pytube` library
- `tkinter` for the graphical file dialog

You can install the necessary Python packages using:

```sh
pip install pytube tkinter
```

### Installation

1. Clone the repository to your local machine:

```sh
git clone https://github.com/yourusername/YVD.git
```

2. Navigate to the project directory:

```sh
cd YVD
```

### How to Use

1. Start the application:

```sh
python youtube.py
```

2. When prompted, enter the URL of the YouTube video you wish to download.

3. A file dialog will open. Select the directory where you want to save the downloaded video.

4. The application will download the video in the highest resolution available and save it to the selected location.
