# SpaceX Launch Viewer (SLV)

The SpaceX Launch Viewer (SLV) is a web application that provides information on SpaceX rocket launches. It categorizes launches into successful, failed, and upcoming missions. The application uses Flask for the backend and Bootstrap along with custom CSS for the frontend.

## Features

- **Launch Categorization**: Displays SpaceX launches in three categories: Successful, Failed, and Upcoming.
- **Launch Details**: Provides detailed information about each launch, including mission name, flight number, launch date, and links to articles, webcasts, and other resources.
- **Responsive Design**: Utilizes Bootstrap and custom CSS for a responsive layout that adapts to various screen sizes.

## Getting Started

### Prerequisites

- Python 3.x
- Flask
- Requests library

You can install the necessary Python packages using:

```sh
pip install Flask requests
```

### Installation

1. Clone the repository to your local machine:

```sh
git clone https://github.com/yourusername/SLV.git
```

2. Navigate to the project directory:

```sh
cd SLV
```

### Running the Application

1. Start the Flask application:

```sh
python app.py
```

2. Open a web browser and navigate to `http://127.0.0.1:5000/` to view the application.
